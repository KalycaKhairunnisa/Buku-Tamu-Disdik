QUICK START:
============
1. Pastikan MySQL sudah running
2. Jalankan: php artisan serve
3. Buka: http://127.0.0.1:8000
4. Mulai menggunakan aplikasi!

STRUKTUR DATABASE:
===================
Table: guest_books
- id (PK)
- kecamatan (string)
- nama_pengambil (string)
- nama_tk_kb (string)
- tanda_tangan (longText - base64 image data)
- created_at (timestamp)
- updated_at (timestamp)

==================

- PHP Version: 8.2+
- Database: MySQL 5.7+
- Node Package Manager: Optional (tidak wajib)
- Frontend: Bootstrap 5 + Font Awesome 6

UNTUK PRODUCTION:
=================
1. Set APP_DEBUG=false di .env
2. Set APP_ENV=production
3. Jalankan: php artisan config:cache
4. Jalankan: php artisan route:cache
5. Setup HTTPS dengan SSL certificate
6. Gunakan production-grade web server (Nginx/Apache)
7. Setup backup database secara berkala

TROUBLESHOOTING:
================
Q: Port 8000 sudah digunakan
A: Gunakan: php artisan serve --port 8001

Q: Migration error
A: Pastikan:
   - MySQL service berjalan
   - .env DB config benar
   - Database sudah dibuat

Q: Signature pad tidak berfungsi
A: Pastikan:
   - Browser support HTML5 Canvas
   - JavaScript enabled
   - Coba di browser lain
